# Study quiz

for [GraphAI Contribution Fes 2025](https://lu.ma/v3ghbeet)

## Stack

- GraphAI
- TypeScript

## Development

### Requirement

- node v20
- gcloud cli

### Setup

- initialize google cloud sdk
  ```
  gcloud init
  gcloud auth application-default login
  ```
- create `.env` file
  ```
  cp .env.template .env
  ```
  And rewrite `.env` file

### Start

```
npm install
npm start
```
